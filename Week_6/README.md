# Week 6 - Employee Portal

A full-stack application built using React (Vite), Express.js, and MongoDB.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB

## How to Run

### Backend
1. Go to the `backend` folder: `cd backend`
2. Install packages: `npm install`
3. Add a `.env` file with `PORT=3000` and `DB_URL=<your-mongodb-url>`
4. Run server: `node server.js`

### Frontend
1. Go to the `frontend` folder: `cd frontend`
2. Install packages: `npm install`
3. Start the Vite app: `npm run dev`

Live Link: https://emp-app-frontend-v2.vercel.app/
