# Week 1 - Foundations

Introduction to basic JavaScript and web concepts.

## Day 1 - Web Architecture & JavaScript Basics
- Learned client-server communication models.
- Explored core JavaScript data types (numbers, strings, booleans).
- Worked with conditionals, loops, and functions.

### Day 1 Assignments
1. Find the larger value between two numbers.
2. Find the maximum value among three numbers.
3. Calculate the sum of elements in an array.
4. Find the minimum element in an array.

## Day 2 - Intermediate JavaScript Concepts
- Explored scope and closures.
- Learned callback functions and asynchronous programming.
- Practiced array and object manipulation.

### Day 2 Assignments
1. Add a new employee at index 1.
2. Delete the employee named "Kiran".
3. Update Sneha's score to 75.
