# Week 1 - JavaScript & Web Foundations 🚀

Welcome to Week 1! This week covers the basic foundations of web architecture and JavaScript. We went from understanding client-server communication to writing our first JavaScript programs.

---

## 📚 What We Learned

### Day 1: JavaScript Basics & Web Architecture
- **Web Concepts:** How the internet works and how clients (browsers) talk to servers.
- **Data Types:** Working with numbers, strings, and booleans.
- **Control Flow:** Writing conditional statements (`if-else`) and loops (`for`, `while`) to control program execution.
- **Functions:** How to write reusable code using standard function declarations and arrow functions.

### Day 2: Intermediate JavaScript & Scope
- **Scoping & Closures:** Understanding how variables are scoped and how closures work.
- **Asynchronous JS:** Using callback functions to manage the order of code execution.
- **Data Collections:** Basic operations on arrays and objects.

---

## 🛠️ Assignments & Practice

### Day 1 Assignments
1. **Compare Two Numbers:** Write a script to find and display the larger of two numbers.
2. **Find Maximum of Three:** Write a script to find the largest among three numbers.
3. **Sum of Array:** Calculate the sum of all elements in an array.
4. **Find Minimum:** Locate the smallest number in an array.

### Day 2 Assignments
1. **Employee Management:** Insert a new employee record at a specific position (index 1).
2. **Filter Employee:** Remove a specific employee named "Kiran" from the list.
3. **Update Record:** Modify an employee's score (e.g., change Sneha's score from 95 to 75).
