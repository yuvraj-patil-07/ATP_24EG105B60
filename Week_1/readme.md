# ATP - Week 1 (Foundations)

This repository contains our initial coursework covering core JavaScript principles and fundamental web architecture.
Over the course of Day 1 and Day 2, we explored fundamental concepts and completed various practical coding challenges.

---

## Day 1 - Web Architecture & JavaScript Basics:

- Explored foundational internet mechanics, specifically client-server communication models.
- Examined core JavaScript data types, including numbers, strings, and booleans.
- Implemented control flow mechanisms using conditional branching and iterative loops.
- Explored function definitions, syntax variations, and modern arrow function declarations.

### Day 1 Assignments:
1. Determine the larger value between two numerical inputs.
2. Identify the maximum value among three given numbers.
3. Calculate the aggregate sum of values within a numerical array.
4. Locate the minimum numerical element present within a list of scores.

---

## Day 2 - Intermediate JavaScript Concepts:

- Investigated lexical scoping rules and the practical mechanics of JavaScript closures.
- Mastered asynchronous workflows using callback functions to manage execution order.
- Studied object and array manipulation techniques to dynamically manage internal state.
- Practiced data operations on structured employee records and updated record properties.

### Day 2 Assignments:
1. Insert a new employee record at index 1 within the employee dataset.
2. Filter out and remove the employee record associated with the name "Kiran".
3. Modify Sneha's recorded score, adjusting the value from 95 down to 75.

---

These exercises establish the necessary baseline skills required for full-stack application development.
