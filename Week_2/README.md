# ATP - Week 2 (Advanced Data Structures)

This module focuses on intermediate JavaScript data collections, exploring sophisticated array and object manipulation techniques.
We implemented several realistic applications, including financial analysis utilities and e-commerce cart algorithms.

---

## Core Curriculum Concepts:

- Processed financial transaction logs utilizing functional array methods such as map, filter, and reduce.
- Constructed a personalized movie suggestion engine guided by user preference profiling.
- Developed dynamic student performance dashboards populated via complex nested object schemas.
- Examined memory references, comparing the mechanics of shallow copying versus deep object cloning.
- Applied spread and rest operators to write concise, immutable data transformation routines.

---

## Primary Implementation Tasks:

1. bankTransactionAnalyzer.js : Compute net cash flow and categorize ledger activity.
2. movieRecomendation.js : Recommend curated titles matching specific viewer profiles.
3. shoppingCart.js : Manage basket state and calculate total order billing totals.
4. employeePayroll.js : Process cumulative compensation metrics across workforce lists.

---

These practical exercises reinforce robust data modeling and functional programming patterns.
