# Week 2 - Advanced Data Structures 📊

This week, we went deeper into JavaScript by working with intermediate data structures, focusing on advanced array methods and object manipulation. These exercises help build real-world logic, such as shopping carts and recommendation systems.

---

## 📚 What We Learned
- **Advanced Array Methods:** Using `.map()`, `.filter()`, and `.reduce()` to easily transform and filter arrays.
- **Copying Objects:** The difference between shallow copying and deep copying, and how they affect memory.
- **Modern JavaScript Syntax:** Using the spread (`...`) and rest operators to write clean, immutable code.

---

## 📂 Projects & Tasks
Here are the files we worked on this week:
- **`bankTransactionAnalyzer.js`**: Analyzes a list of bank transactions to compute net cash flow and categorize spending.
- **`movieRecomendation.js`**: A basic movie recommendation engine that suggests films matching user profiles.
- **`shoppingCart.js`**: Simulates an e-commerce shopping cart, handling items, quantities, and calculating the final price.
- **`employeePayroll.js`**: Processes payroll details and calculates metrics across a workforce list.
