# ATP - Week 3 (Server Architecture & Databases)

This module introduces server-side application development utilizing Node.js, Express, and MongoDB.
We executed database queries and constructed a RESTful API service to manage user data.

---

## Core Learning Objectives:

- Explored NoSQL database architecture, structuring records into MongoDB collections and documents.
- Practiced essential CRUD workflows (Create, Read, Update, Delete) via the mongo interactive shell.
- Built an Express.js web server configured to intercept HTTP incoming GET and POST requests.
- Established secure remote database connections to MongoDB Atlas cloud clusters via connection strings.

---

## Directory Index:

- MongoDbPractice.txt : Reference log of raw query commands executed within the database terminal.
- server.js : The primary Express server entry point handling incoming network traffic.
- models : Directory containing Mongoose schemas defining structured user and product entities.
- reqq.http : REST Client test suite verifying HTTP request and response payload correctness.

---

Backend engineering provides immediate visibility into data flow and network request lifecycles.
