# ATP - Week 4 (HTML & CSS Styling)

This module covers semantic web markup and visual presentation principles using HTML5 and CSS3.
We structured document layouts and applied custom cascading stylesheets to achieve responsive designs.

---

## Core Curriculum Topics:

- Explored semantic HTML tags, constructing accessible document outlines with headings and text blocks.
- Investigated the box model hierarchy, managing container nesting and structural element relationships.
- Developed a dedicated media showcase page incorporating embedded responsive images and hyperlinks.
- Applied CSS styling rules to customize typography, color palettes, margin distributions, and padding.
- Implemented interactive form components capturing user input across structured input fields.

---

## Directory Contents:

- HtmlElements.html : Demonstration document featuring a wide array of baseline HTML tags.
- NestedHtmlElements.html : Layout exercise emphasizing DOM tree nesting and container styling.
- Cat.html : Themed web presentation highlighting responsive media embedding and anchor tags.
- style.css : Master external stylesheet defining visual aesthetics and layout alignments.
